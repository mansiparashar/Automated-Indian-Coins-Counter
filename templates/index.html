Coin Classification Application

A Flask Application backed by Artificial Intelligence
for automatic classification of coins, using Image Processing.

Let's Get Started

Know our application

How does this app work?

An image can be uploaded on the web app. This image, in the backend undergoes numerous Image Processing techniques are applied as pre processing. These include resizing, thresholding, object detection,histogram equivalence,HOG and more.

We use our trained CNN model to call the predict function on this processed image, and returns the classification on the front end.Flask is used as the backend, with HTML/CSS for a responsive front end application.

Image Processing techniques along with deep learning. A computer Vision project.

A web application which tells you the value of indian coins in an image.
